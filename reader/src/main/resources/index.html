Maven Easy Index

(v0.0.5)

This webapp is Maven Easy Index, a library and REST API that provides simplified access to the Maven Central Index.

The Maven Central Index describes all changes in the repository, like added or deleted packages. Updates are released in incremental, consecutively numbered index files and this webapp provides an easy interface to access the index contents. The relevant information from the index is exposed via a REST API.

In addition to this instruction page, two endpoints can be used to conveniently access the contents of index files by providing their file number.

  • exists/«int» (link uses id 457)
    200 (OK):
    the indicated index exists
    404 (NOT FOUND):
    the indicated index does not exist
  • get/«int» (link uses id 457)
    200 (OK):
    response contains a JSON array of all contained artifacts
    404 (NOT FOUND):
    the indicated index does not exist
